# spedit

## overview

sprite editor  for HSP3.

入力ファイルフォーマット

・BMP file（24bit/4bit color）

・NES file（CHRエリアだけを読み取ります）

・CHR file（ファミコン用2bit color）

出力ファイルフォーマット

・.BMP

・.h/.c/.bin（PCエンジン用4bit color/スーパーファミコン用4bit color/ゲームボーイ用2bit color）

・CHR（ファミコン用2bit color）
